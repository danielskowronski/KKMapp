KKMapp is unofficial app for checking active Cracov's municipal transport tickets (MPK) stored on KKM (Krakowska Karta Miejska /Cracov City Card).

This is open-source app - chceck https://github.com/danielskowronski/KKMapp
Author: Daniel Skowroński <daniel@dsinf.net>, http://dsinf.net

*check active ticket on KKM

This is very first release with basic functionality only - stay tuned for pinning tickets tiles to start menu, reminders and many more ;)
